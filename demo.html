🎯 How to Test File Sharing

Since WebRTC requires two separate peers, you'll need to:

  1. Open two browser tabs/windows with the application
  2. Tab 1 (Sender): Click "Create Offer (Sender)"
  3. Copy the offer from Tab 1
  4. Tab 2 (Receiver): Click "Join Session (Receiver)"
  5. Paste the offer in Tab 2 and click "Set Offer"
  6. Copy the answer from Tab 2
  7. Paste the answer in Tab 1 and click "Set Answer"
  8. Wait for connection - you'll see "Connected!" message
  9. Select files in Tab 1 and click "Send Selected Files"
  10. Download files in Tab 2 when they appear

✨ Features Demonstrated

  • 🔐 Secure WebRTC Connection - DTLS encrypted
  • 📁 File Transfer - Any file type, any size
  • 📊 Progress Tracking - Real-time transfer progress
  • 📋 Easy Signaling - Copy/paste offer and answer
  • 💾 Automatic Download - Files ready to download on receiver

🛠 Technical Highlights

  • Pure JavaScript - No frameworks or dependencies

  • File Chunking - 16KB chunks for reliable large file transfer
  • Error Handling - Comprehensive validation and user feedback
  • Cross-Browser - Works in Chrome, Firefox, Safari, Edge
